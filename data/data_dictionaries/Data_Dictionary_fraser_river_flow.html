Historical Hydrometric Data Download

Daily Data

Comma Separated Values

A time series per available parameters, with a header row between each time series.

Column: ID

The unique 7-character station identification, in the format ddaaddd, where

  • dd: 2 digits for the major drainage basin
  • aa: 1 alphabetic character each for the sub-basin and sub-sub-basin
  • ddd: 3 digits for the specific station within the sub-sub-basin
Column: PARAM

The parameter codes, where

  • 1: discharge (flow)
  • 2: water level
Column: TYPE

The data type, where

  • 1: values in m3/s)
  • 8: values in m (to mm precision)
  • 9: values in m (to cm precision)
Column: YEAR

The year, in the format yyyy.

Column: DD

The day of the month, in the format dd.

Columns: JAN, FEB, MAR,AVR,MAR,JUN,JUL,AUG,SEP,OCT,NOV,DEC,SYM

The values. The associated columns SYM may be empty or contain the following

  • A: partial day
  • B: ice conditions
  • D: dry
  • E: estimated
  • S: sample(s) collected this day

Station Information

Comma Seplarated Values

The metadata of the station.

Column: Station Number

The unique 7-character station identification, in the format ddaaddd, where

  • dd: 2 digits for the major drainage basin
  • aa: 1 alphabetic character each for the sub-basin and sub-sub-basin
  • ddd: 3 digits for the specific station within the sub-sub-basin
Column: Station Name

The name of the station. This field is enclosed in quotes since it may contain commas.

Column: Province

The province code where the station is located, where

  • AB: Alberta
  • BC: British Columbia
  • MB: Manitoba
  • NB: New Brunswick
  • NL: Newfoundland and Labrador
  • NS: Nova Scotia
  • NT: Northwest Territories
  • NU: Nunavut
  • ON: Ontario
  • PE: Prince Edward Island
  • QC: Quebec
  • SK: Saskatchewan
  • YT: Yukon Territory
Column: Status

The status of the station, where

  • A: active
  • D: discontinued
Column: Latitude

The latitude of the station, in decimal degrees.

Column: Longitude

The longitude of the station, in decimal degrees.

Column: Year From

The first year for which there is data available, in the format yyyy.

Column: Year To

The last year for which there is data available, in the format yyyy.

Column: Drainage Area

The gross drainage area covered by the station, in km2.

Column: Sediment

A flag indicating if there is a sediment program at the station, where

  • Y: yes
  • N: no
Column: RHBN

A flag indicating if the station is part of the RHBN, where

  • Y: yes
  • N: no
Column: Real-Time

A flag indicating if the data is available in real-time, where

  • Y: yes
  • N: no
Column: Datum Name

The name of the datum used.
